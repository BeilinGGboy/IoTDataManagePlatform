# 阿里云安全组配置 - 放行 8080 端口

---

## 一、安全组是什么？

安全组相当于云服务器的**防火墙**，默认只放行少量端口（如 22）。要外网访问 8080，必须在安全组里添加规则。

---

## 二、配置步骤（图文）

### 1. 登录阿里云控制台

打开 https://ecs.console.aliyun.com/

### 2. 进入实例列表

左侧菜单 → **实例与镜像** → **实例**

### 3. 找到你的实例

找到公网 IP 为 `47.100.174.12` 的实例，点击实例 ID 进入详情。

### 4. 进入安全组

在实例详情页，找到 **安全组** 这一行，点击安全组 ID（如 `sg-xxxxx`）。

### 5. 配置入方向规则

- 点击 **入方向** 标签
- 点击 **手动添加** 或 **添加规则**

### 6. 填写规则

| 字段 | 填写值 |
|------|--------|
| 授权策略 | 允许 |
| 协议类型 | TCP |
| 端口范围 | 8080/8080 |
| 授权对象 | 0.0.0.0/0 |
| 描述 | 应用服务（可选） |

### 7. 保存

点击 **保存**，规则生效通常很快（几秒内）。

---

## 三、快速验证

配置完成后，在浏览器访问：

```
http://47.100.174.12:8080
```

能打开管理平台页面即表示配置成功。

---

## 四、常见问题

| 问题 | 处理 |
|------|------|
| 找不到安全组 | 在实例详情页，安全组在「安全组」或「网络与安全」区域 |
| 已有 8080 规则 | 检查是否在「入方向」，且授权对象为 0.0.0.0/0 |
| 仍无法访问 | 检查服务器防火墙是否放行 8080（见下方） |
